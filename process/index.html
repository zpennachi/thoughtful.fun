  1. Adobe Medium Sculpt: The main tool used to create the starting block/main visual component of my workflow.
  2. Blender Optimization:
    • Meta Spark Toolkit: A little Blender plugin that decimates and optimizes the geometry for glb format. I think it's just the same as the regular decimate modifier, but it has a few extra quirks, and I’ve just used it for so long I'm more comfortable with it than the standard modifier.
  3. Modelviewer Editor: To quickly make sure my model works and to make some small tweaks on my materials if needed!
    • While it's in this “final form” of sorts, I’ll screengrab some different angles/zoom levels for the print aspect of my process.
    • Export: I’ll then click “download scene” and drag out the glb file to my desktop. This will be the final file I upload to my site after I compress it.
  4. GLB Compressor: This tool crunches down my model efficiently. I’m probably giving them all of my 3D models, but I don’t mind, haha.
    • With this final file, I’m now ready to get it up on my website!
  5. Dithermark: I love Dithermark; its endlessly tweakable interface makes this a nice meditation in subtle evolutions. I’ll upload the various images from the Modelviewer screengrabs (step 3).
  6. Figma: I bring all of the Dithermark content into Figma to create my final print designs. I use Figma's typography alongside the visuals created through the process so far. I’ve got existing templates for a few different-sized stickers I print and can just ctrl + drag to duplicate them. So fast, so easy!
  7. Image Compression: I’ll take all of the final images I’m using for the website and compress them before uploading, just to squeeze out every last bit before getting them on the web!
